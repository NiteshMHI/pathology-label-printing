# Pathology Barcode Label Printing System

A desktop application for generating and printing pathology barcode labels. The application supports printing a **set of 6 barcode labels** on a single print job using any compatible thermal label printer.

## Features

* Print **6 pathology barcode labels** for each sample.
* Supports **2-Up (2 labels per row)** layout.
* Label size: **38 × 25 mm**.
* Generates high-quality barcode labels.
* Fast printing for laboratory workflows.
* Designed for offline use.

## Supported Label Printers

The application is compatible with most thermal label printers, including:

* TSC
* Zebra
* Citizen
* SATO
* And other printers supporting **38 × 25 mm** labels.

## Label Specifications

| Property       | Value                         |
| -------------- | ----------------------------- |
| Label Size     | 38 × 25 mm                    |
| Layout         | 2-Up (2 Labels per Row)       |
| Labels per Set | 6                             |
| Printer Type   | Thermal Barcode Label Printer |
